## Description: 
     The goal of this project was to create an employee management system that is easy to use for non-developers using node, inquirier and MySQL. Users are able to view employees/roles/departments, add employees/roles/departments and update employee roles.
## Installation:
     To install this program, you have to run `npm install`, followed by `node start` in the console.
## Usage:
     This project should be manage an employee database.
